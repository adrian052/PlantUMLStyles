# PlantUMLStyles
IOHK inspires PlantUML Styles

<h1>How to Use

```plantuml
@startuml
!define TESTNETLIGHT
!includeurl https://raw.githubusercontent.com/adrian052/PlantUMLStyles/master/iohkStyle.puml
' your UML '
@enduml
```
